如果您不同意按照本隐私政策所述方式处理您的个人信息，请停止浏览本博客。继续浏览本博客即表示您确认本站在本隐私政策中所述的有关您个人信息的规定。

# 浏览

请您悉知，当您浏览本博客时，我们会收集您的必要的相关信息（IP 地址、浏览器的 User Agent 等）用于识别是否为恶意攻击。

当您从中国大陆以外的国家 / 地区或者通过 IPv6 的方式访问本博客时，将会通过 Cloudflare 进行接入，详情请参考 其他网站的嵌入式内容 。

# 评论

当您留下评论时，我们会收集您的相关信息（IP 地址、浏览器的 User Agent 等）来帮助识别是否为恶意攻击以及留存证据以供必要时的司法取证。

当您留下评论后，您可能会收到相关邮件通知（评论回复通知，网站变更通知等）。

当您留下评论后，我们会展示出您当前评论所使用的计算机系统名称，例如 Windows 10 。同时，我们也会展示出您当前评论所使用的浏览器名称以及大致版本，例如 Edge 124 。

当您使用注册功能时，我们会收集您的相关信息（您主动填写的昵称、邮箱、网址、密码、注册时间等）来为您注册账户。

当您使用登录功能时，我们会收集您的相关信息（您主动填写的邮箱、密码等）来验证您是否有账户使用权。

在未征得您的同意前，我们不会将您的地理位置公开展示，更不会将您的 IP 进行公开。但请注意，这并不包括您因恶意攻击本站而被记录下的地理位置和 IP 地址。

# Cookies 与 Local Storage

当您留下匿名评论时，我们会将您所填写的「昵称」，「电子邮件」，「网址」通过 Cookie 技术和浏览器本地存储功能保留在您的计算机上，以便于您之后的评论。

我们不会强制要求您使用 Cookie，您可以关闭您的浏览器的 Cookie 功能或者手动清除相关 Cookie。

请您悉知，本博客包含其他其他网站的嵌入式内容，这些网站可能会收集您的信息，使用 Cookies 嵌入额外的第三方跟踪程序。
