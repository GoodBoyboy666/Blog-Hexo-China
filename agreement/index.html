用户协议与隐私政策

# 用户协议

当您继续浏览本站时，即代表你已同意本用户协议。

# 内容

为维护本博客的知识产权，与您约定本用户协议。

本站内容包括但不限于文章、图片、音频、视频、CSS、JS 以及本站内除评论的其他由本站制作的内容。本站使用的主题版权归主题作者所有。

本站内容无特殊声明均以 CC BY NC-SA 4.0 授权发布。您可以在每篇博文或页面底部了解该篇博文的版权信息以及授权方式。

# 评论

在本站评论时，请确保您的评论符合以下规则，否则本站会视情况调整评论内容包括但不限于折叠，屏蔽等，或直接删除您的评论：

  • 评论内容应有意义、参考价值，不得灌水刷屏等；
  • 昵称，邮箱，网址，评论内容不得包含任何商业推广信息，不得包含包括但不限于欺诈链接，黄色网站；
  • 昵称，邮箱，网址，评论内容须符合中华人民共和国相关法律法规，不得包含任何令人不适的内容。

您对您的评论内容拥有完全的著作权。同时，您也需要对您所发布的评论承担全部责任。

# 免责声明

您对您发布的评论承担全部责任。如果您的评论因包括但不限于真实性，有效性，安全性等原因损害了他人利益，本站不承担任何责任。

本站公开的内容包含大量主观性内容，因此本站不保证其内容的客观性。

本站公开的内容均经过我们的审核，如果您因个人原因或使用非法方式访问本站非公开内容包括但不限于隐藏内容，加密内容所造成任何损失，本站不承担任何责任。

# 隐私政策

如果您不同意按照本隐私政策所述方式处理您的个人信息，请停止浏览或使用本博客。继续浏览本博客即表示您确认我们在本隐私政策中所述的有关您个人信息的规定。

# 浏览 / 使用

请您悉知，当您浏览或使用本博客时，我们会收集您的必要的相关信息包括但不限于 IP 地址、浏览器的 User Agent 等必要信息用于识别是否为恶意攻击。

当您从中国大陆以外的国家 / 地区访问本博客时，可能通过 Cloudflare 访问本博客，详情请参考 其他网站的嵌入式内容 。

# 评论

当您留下评论时，我们会收集您的相关信息包括但不限于 IP 地址、浏览器的 User Agent 等必要信息来帮助识别是否为恶意攻击以及留存证据以供必要时的司法取证。

当您留下评论后，您可能会收到相关邮件通知包括但不限于评论回复通知，网站变更通知等。

# Cookies

当您使用本博客时，我们会通过 Cookie 保存相关设置以确定我们是否需要通过浏览器向您推送新文章。

当您留下评论时，我们会将您所填写的「昵称」，「电子邮件」，「网址」通过 Cookie 技术保留在您的计算机上，以便于您之后的评论。

我们不会强制要求您使用 Cookie，您可以关闭您的浏览器的 Cookie 功能或者手动清除相关 Cookie。

# 其他网站的嵌入式内容

请您悉知，本博客包含其他其他网站的嵌入式内容，这些网站可能会收集您的信息，使用 Cookies 嵌入额外的第三方跟踪程序。

本博客包含的第三方嵌入内容清单：

  • Jsdelivr CDN（用于加速网站访问）
  • Webpushr（用于浏览器推送）
  • Bing（用于网站背景图）
  • Gravatar（用于获取您在 Gravatar 设置的头像）
  • Cravatar（用于获取您在 Cravatar 设置的头像）
  • hitokoto（用于获取一言）
  • Umami（用于网站流量统计）
  • GeeTest（用于评论人机验证以及网站安全）
  • Cloudflare（用于网站全球 CDN 加速以及网站安全）
  • busuanzi（用于文章浏览量统计）
  • 博客朋友圈（用于朋友圈页面的文章获取）
  • 友情链接头像（用于显示友情链接页面其他站长的头像）

# 保存时长

我们将无限期保存您的信息，直到您向我们正式提出删除您的个人信息的申请。

# 信息共享、转让

我们不会向第三方共享、转让您的个人信息，除非经过您本人事先授权同意。

注：您与第三方网站（其他网站）交互所被收集的个人信息不在我们的管理范围内。

# 跨境传输

请您悉知，您的个人信息可能会被跨境传输至中国大陆以外的国家 / 地区。

# 服务商

我们采用云服务商提供的服务来托管我们的各项服务。请您悉知，您的数据可能会托管于以下云服务商：

  • 阿里云（https://www.aliyun.com/）
  • Azure（国际）（https://azure.microsoft.com/）
  • CloudFlare（https://www.cloudflare.com/）
  • Vercel（https://vercel.com/）
  • Netlify（https://www.netlify.com/）
  • Upstash（https://www.upstash.com/）

# 豁免征得同意收集和使用的个人信息

请您理解，在下列情形中，根据法律法规及相关国家标准，我们收集和使用您的个人信息无需征得您的授权同意：

  1. 与国家安全、国防安全直接相关的；

  2. 与公共安全、公共卫生、重大公共利益直接相关的；

  3. 与犯罪侦查、起诉、审判和判决执行等直接相关的；

  4. 出于维护个人信息主体或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；

  5. 所收集的您的个人信息是您自行向社会公众公开的；

  6. 法律法规规定的其他情形。

# 您的权利

您可以向我们提出删除您的个人信息的申请。

如果您提出删除您的个人信息申请，在我们完成相关检查流程后，我们将会正式删除您的个人信息。但请您悉知，这并不包括我们因相关法律法规或安全需要所必须保留的数据。

# 变更

请您悉知，我们可能会适时对本用户协议和隐私政策进行修改。

如果本用户协议和隐私政策有任何变更，我们将在此发布更新的用户协议与隐私政策。请定期访问本页面，查看本用户协议与隐私政策是否有任何更新或变更。

  • 其他网站、第三方网站：指域名不同于 blog.goodboyboy.top 的网站。
