第七篇 选择网站域名 | 从零创建一个属于自己的网站(小白向)

发表于2023-09-01|更新于2023-09-01|教程Web
|字数总计:1.8k|阅读时长:5分钟|阅读量:

# 前言

这一篇也是比较个性化的，根据自己的喜好和实际情况来选择吧

# 什么是域名

直接搬百度百科–> 域名

域名的实际作用是为了便于访问目标服务器，因为十进制的 ip 一般都很难记忆，所以域名的最基础作用就是与 ip 绑定便于用户访问

由此可见，域名就像一个人的名字一样，你叫这个人名字就等同于你叫这个人一样

当然域名的作用不仅仅就是充当个名字，这个我们之后再介绍域名的其他玩法

# 注册域名

# 起一个自己喜欢的 “名字”

这个就看自己的喜好了，你可以把寓意全部都放在域名的位置，也可以联合域名后缀一起完成全部寓意

例如 loveu.com 或者 eve.online（确信😂

总之看你自己的喜好吧，域名后缀也有很多，具体能用的后缀根据注册商而定

注：最好不要用全数字的那种，因为黄色，赌博之类的网站最喜欢这样用，容易出现乌龙（doge

不过有些词可能因为某些原因变成溢价域名，溢价域名一般都很贵

一般来说，字母小于 3 个的域名都会溢价，或者超级贵，所以请根据自己的财力选择

# 选择注册商

这里注册商分国内和国外

这两个区别最大的地方在于，国内注册商应国家要求，所有域名都需要实名认证，而国外域名注册商则不强制要求

而且一些后缀只有国内域名注册商才有，例如.cn ,.com.cn 之类的

像 cn 这种国家域名级在国外注册商是注册不了的，gov.cn 那就更不用说了，一般人都注册不了，专属于国家机关

同时，如果域名在国外注册想要转入国内，也是需要在转入的注册商完成实名认证

国内注册商一般像阿里云，腾讯云，华为云，百度云，西数之类的，都挺多的

国外的像 aws，namesilo，namecheap（好像听说有口碑问题），cloudflare，Google 等等一大堆

不同的注册商同一个域名后缀可能价格有所差异，就看自己的需求以及能力了（有些注册商需要使用 visa 或者万事达信用卡支付，不支持国内的支付宝微信和银联支付）

# 开始注册

在域名注册页面可以查询到该域名的状态，是未注册还是已注册，以及它的价格是多少

一切都准备好后可以就可以进行支付注册了，国内注册的可能会提醒你进行实名认证（如果你是第一次新号注册的话）

注册完成以后需要等待 ns 解析生效（name server），一般可能需要等待 1-2 个小时，全球生效可能需要等待 1-2 天

部分注册商在域名注册完成后会默认使用自家的 ns，有些注册商则在注册时会询问

一般没有特殊需求使用注册商的 ns 即可

国内和国外的 ns 也是有区别的，国外 ns 提供的 dns 解析服务一般是分不了线路的，国内 ns 提供的 dns 解析服务一般都支持分线路，例如电信，联通，移动，教育网，境外等等（高级版可能还可以更细致），不同的线路可以设置不同的解析

所以根据自己的实际情况来选择即可

# 设置 dns 解析

# 记录类型

一般我们用的最多的是 A 解析和 CNAME 以及 TXT 解析（多用于申请 SSL 证书）

这里解释一下部分记录的作用

记录 解释
A 记录 一般用于将域名指向 IPv4 地址
CNAME 记录 一般用于将域名指向另一个域名
MX 记录 一般用于将域名指向邮件服务器
AAAA 记录 一般用于将域名指向 IPv6 地址
TXT 记录 一般用于标识和说明，可用来在申请 SSL 证书时的 DNS 验证
URL 转发 一般用于域名的重定向

更多的记录请参考 ns 服务商的文档

# 主机记录

一般都会设置 www 和 @（用作 CDN 的域名可能不会设置）

至于其他的就看自己的实际需求了

# 记录值

填写想要解析的目标 ip 或者域名

# 解析线路

没有特殊需求默认即可

# TTL

没有特殊需求默认即可

# 隐私问题

域名注册后是可以查询到 whois 的，也就是注册人的信息

在国外注册的域名可能无所谓（毕竟没有强制要求实名认证），但在国内注册的域名如果没有开启域名 whois 隐私，则你实名认证的所有信息都会被查出来

包括但不限于姓名，联系电话，家庭详细住址等等

所以如果是国内注册的域名请确保已开启 whois 隐私（现在一般都会默认免费开启）

但是部分注册商的 whois 隐私不完善，可能仍然会泄露出部分个人信息，这点需要注意

cn 后缀的域名 whois 隐私截止于写稿前是需要单独付费的，不同注册商价格可能会有差异！！！

# 更多玩法

# 域名邮箱

像 xxx@qq.com xxx@163.com 之类的电子邮件地址，相信大家肯定都发现了吧，他们的后缀都是域名，那我们是否也能这样玩呢？答案是肯定

这种玩法在国内一般叫企业邮箱，你可以自己搭建邮件服务器进行托管，也可以使用大厂的服务，阿里云就提供有免费的企业邮箱

一般来说，选择大厂的企业邮箱要好过自己搭建，因为考虑到稳定性以及连通性，大厂的服务肯定比自己搭建要好很多

# 申请 SSL 证书

有域名之后就可以给域名申请 SSL 证书了，现在普通的 DV 证书都有免费的可供选择，部分 web 面板也有一键申请的功能，当然也可以自己去服务商那里进行申请

其实如果没有域名也是可以申请 SSL 证书，不过 ip 的 SSL 证书基本是没有免费的而且价格不便宜，所以申请域名的 SSL 证书更划算

# 解析各种 ip

既然可以将域名解析到 ip，那肯定是不止于玩网站了，像各种游戏服务器，例如 MC，比直接输入 ip 或者一大串的域名要高大上很多（doge

如果自己家庭宽带有公网 ip，你也可以解析到家庭宽带，做好端口映射，方便访问内网各种服务等等

# 后记

本篇教程差不多就这样子了，更多玩法还需要大家继续探寻哦～
