办了张新卡惹

发表于2024-03-21|更新于2024-03-21|说说
|字数总计:1.3k|阅读时长:3分钟
文章摘要
GoodBoyboyGPT
文章内容为一位作者办理了招商银行的哔哩哔哩联名花嫁卡的经历。通过手机 APP 申请、制卡、领卡等流程，描述了取卡过程中的细节和限额情况。同时提到新客户权益以及对其他银行开卡经历的比较，表达了对现有政策放松和不同地区规定的感慨。最后分享了自己在建行开卡时遇到的困难和挫折，展示出办理银行卡并非易事。
此内容根据文章生成，并经过人工审核，仅用于文章内容的解释与总结

# 前言

之前在网上就看到了招商银行的哔哩哔哩联名的花嫁卡，而且还是储蓄卡，再加上招行的服务真的超级棒，于是果断申请。

# 申请

在招商银行 APP 上选择 “申请储蓄卡”，然后填写相关信息就可以了。

接着就是等着制卡。从审批到制卡完成大概一个星期左右。

# 制卡

说是制卡，但是当时我去领卡的时候招行的小姐姐是在有个智能发卡机上直接取的卡，并不是我想象中的那种情况（意味着可能那台机器里本来就有这种卡面的卡）。

但是取卡之前小姐姐又问我有没有收到领卡的短信，就有点小迷惑。

而且取卡的时候小姐姐是叫我自己在那台机器上找我申请的卡面，也不是说像那种直接查询出你申请的卡面。

当时我其实还在想既然是自己找要不干脆临时换个卡面？（毕竟招行储蓄卡好看的卡面太多了，但是现在一类卡又只能开一张……）

# 领卡

领卡过程倒是蛮顺利的，流程很简单，就首先扫个反诈信息的二维码，填个身份证信息，然后写个那个类似于保证的单子（就是那个 “我已阅读并……” 的单子）。

接着在大厅的沙发上等了一会，又来了个小姐姐就带我去智能终端机器上进行登记，领卡。

最后拿到卡绑定微信支付宝测试了一下大额，小额存取，没问题后就完成了。

# 限额

因为职业身份是学生的原因，新开的卡系统默认限制非柜面交易每天账户流出为 1000 元，小姐姐也无法修改，但可以等一个月后再在 APP 上自行修改。

不过我倒是无所谓，反正我卡多，各种限额额度的都有，而且本来就打算用这张卡来绑微信和支付宝进行日常小额消费。

# 权益

又是该死的小字。

新客户首开花嫁卡可以领取哔哩哔哩大会员季卡，当时我选择花嫁卡也是有这个成分在里面，结果当我拿到卡准备领取的时候说我不符合要求，然后我就发现对于 “新客户” 有一行小字说明，大致意思是：新人第一张卡是花嫁卡才能是 “新客户”，如果之前有一网通之类的账户就不算 “新客户”。

我之前在云闪付上面申请了一个二类的一网通账户（因为卡面是皮卡丘的超好看，不过没实体卡），所以就不算 “新客户” 了，麻了……

# 感想

不知道这是招行特色还是说现在政策放松了。

记得之前因为断卡行动，本地开卡都难得要命，更别说异地开卡了（我家那边没招行网点，我读书的这边才有）。

当时高中毕业我自己开一张建行卡麻烦的要死，光光单子就填了三张，还要求录取通知书复印件，当我填完后结果来办卡的大学生太多，银行柜员建议我去另外一个网点……

于是我又拿着那三张单子跑到了另一个网点办。

本以为这就完了，结果历史遗留问题又开始攻击我了……

在建行我名下因为已经有了一张一类的建行卡，所以办不了，麻了……

那个一类卡是我 16 岁的时候在建行官网时申请的，申请已经批下来了，卡也发到了那个网点，结果网点以我未成年就不给我办。

这个我倒是没什么意见，毕竟每个地区政策不同（有些地区 16 岁可以自行办银行卡），但是无语的是它没销卡…… 也就是说账户已经成功建立了。

以至于要办一类卡就要把原先那张卡给注销掉，但是他们好像销卡必须要收回实体卡（但是我 tm 压根就没有），柜员也无语了……

最后折中给我办了个二类实体卡（蚌埠住了）。

后来我二类实体卡用着别扭，又去网点打算试试把那张 “蜜汁” 一类卡注销掉，然后就是迎接的人叫我取号去柜台办理，叫到号后柜台的人一看又叫我去迎接的人那里让她带我去机器那里办理……

这经典踢皮球踢得我猝不及防。

最后那张 “蜜汁” 一类卡还是没有被销掉，只是从一类卡降到了二类卡，把我的实体二类卡升级成了一类卡。

那张 “蜜汁” 卡至今还挂在我名下……

可以看出确实是没有对比就没有伤害。

# 好看的卡
