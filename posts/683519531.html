# 前言

前段时间用自己的教育邮箱白嫖了微软 1T 的 OneDrive 空间，结果在本地挂载的时候发现因为主域名无管理员（也就是学校没弄管理员账号），因此无法使用 API 挂载，导致无法正常映射到本地。

时隔多日在今天再次折腾时发现可以用 rclone 的 WebDav 挂载。

# 后记

听说使用这种方式挂载好像性能和稳定性不是很好？不过对于我来说能用就行。
