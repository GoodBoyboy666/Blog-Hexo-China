我的世界皮肤站披风不加载或不更新问题

发表于2021-01-31|更新于2021-01-31|教程Web
|字数总计:493|阅读时长:1分钟|阅读量:

皮肤站不加载或不更新披风有很多情况，需要一一来排查。

# 1. 首先检查在游戏中是否穿戴了鞘翅

如果你的披风材质图片没有包含鞘翅的材质部分，安装鞘翅后披风会默认隐藏

解决方法：取下鞘翅或者修改披风文件

# 2. 检查皮肤站是否关闭，或者出现 500 等错误

这个时候 minecraft 是无法正常请求到披风文件，所以就不会加载

解决方法：如果不是站长，就换个皮肤站，站长的话就启动皮肤站，500 错误有很多情况，这里不一一阐述，请大家自行百度。

# 3. 检查是否正确使用了皮肤站

请确保你的披风已经应用于你的角色

解决方法：应用即可

# 4. 检查是否能正常请求到用户材质

先访问 皮肤站域名 / 你的游戏用户名.json 例如 https://skin.goodboyboy.top/goodboyboy.json

看是否能正常请求。成功的话应该是一串 json，再在这串 json 中找到 cape 后面的一串 ID，访问 皮肤站域名 /csl/textures/ 刚刚 cape 的 ID 如果成功的话就会出现你的披风源文件。如果刚刚两项中任何一项出错就是皮肤站出问题了

解决方法：百度、询问作者、重装皮肤站

# 5. 如果上面都没问题，现在你需要检查 mod

检查你是否正确安装并加载 CustomSkinload 皮肤 mod（有可能是你的 mod 加载出了问题）

解决方法：正确加载（并且不要忘记把皮肤站的皮肤加载地址添加到配置文件）

# 6. 检查是否开启了缓存功能

检查.minecraft\CustomSkinLoader\CustomSkinLoad.json 里面 cacheExpiry 改为 0，enablecacheautoclean 的 false 改为 true，删掉在.minecraft\CustomSkinLoader\caches\ 下面的文件。最后重新连接服务器即可（需要多重复几次）
